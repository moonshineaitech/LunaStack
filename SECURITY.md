# Security Policy

## Reporting a Vulnerability

If you discover a security issue in LunaStack (e.g., a protocol that could cause harmful behavior, a skill that leaks sensitive data, or a supply chain concern), please report it responsibly.

**Email:** [open a private security advisory](https://github.com/moonshineaitech/LunaStack/security/advisories/new) on GitHub.

Do **not** open a public issue for security vulnerabilities.

## Scope

LunaStack is a collection of AI prompt protocols (Markdown files). Security concerns include:

- **Prompt injection:** A protocol that could be manipulated to override safety constraints
- **Data exfiltration:** A skill that instructs the AI to send data to external services
- **Malicious skills:** Community-contributed protocols with hidden harmful instructions
- **Supply chain:** Compromised dependencies in setup scripts

## Response

We will acknowledge reports within 72 hours and aim to resolve confirmed issues within 7 days.
